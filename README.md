This repository contains code for performing robust dose-response analysis.
robustDRCfunctions.R contains functions to compute robust estimators for dose-response models.
derivatives.R contains functions to compute robust confidence intervals for the parameters of a dose-response models with a 4-parameter-logistic regression function.
generate.R contains code to generate dose-response data
data_analysis.R contains code to perform data analysis for a concrete example of dose-response data
All the csv files contain the data generated by an experiment performed in the Faculty of Exact an Natural Sciences of the University of Buenos Aires, Argentina, to assess the combined citotoxicity of agrochemicals.
